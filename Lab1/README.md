# Lab1 Project

## Опис
Lab1 — це консольний додаток на C#, який виводить привітання у консоль. Проєкт містить клас `Hello`, що відповідає за генерацію повідомлення.

## Структура
- `Program.cs` — містить точку входу та клас `Hello` для виведення повідомлення.
- `README.md` — містить інформацію про проєкт.

## Запуск
Щоб запустити програму, скомпілюйте та виконайте її за допомогою .NET CLI:
```
dotnet run
```

## Вимоги
- .NET SDK
- С# 10 або новіша версія
